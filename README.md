# Catch JSONL Parser

## Prerequisites
- [Node](https://nodejs.org/en/download/) v14
- [Yarn](https://classic.yarnpkg.com/en/docs/install#install-via-npm) (Optional)

## Installation
```
# if yarn installed
yarn

# or use npm
npm install
```

## How to Use
```
# if yarn installed
yarn start

# or use npm
npm run start
```
